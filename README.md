# ReScript Project

- [Installation](../../README.md)

Official ReScript starter template.

## Installation

```sh
npm install
```

## Build

- Build: `pnpm run res:build`
- Build & watch: `pnpm run res:dev`

- Build Parcel: `pnpm run build`

## Run

```sh
pnpm run start
```

## Clean

- Clean: `pnpm run res:clean`
